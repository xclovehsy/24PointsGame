# 24PointsGame

## 项目简介

已知一副扑克牌有54张，去除大王和小王，剩余52张。在其中抽取4张牌，利用加减乘除进行计算得到24。编写程序生成一副扑克牌，抽取4张，进行计算是否能得到24. 如果可以，列出可能的计算表达式，可能有多种计算形式
